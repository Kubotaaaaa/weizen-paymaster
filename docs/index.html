⚖️ Paymaster Types

  • เซ็น signature off-chain เพื่ออนุมัติว่าจะสปอนเซอร์ UserOp นี้
  • ผู้ใช้จ่าย 0 ETH — paymaster ออกค่าแก๊สให้ทั้งหมด
  • เหมาะกับ onboarding / gasless UX ที่ฝั่งแอปเป็นคนตัดสินใจว่าใครได้สปอนเซอร์
  • ผู้ใช้จ่ายค่าแก๊สเป็น ERC-20 (เช่น USDC) ไม่ต้องถือ ETH
  • Paymaster ออก ETH ก่อน แล้ว postOp ดึง token + swap คืนเป็น ETH
  • ต้องอ้างราคา token/ETH (oracle) เผื่อ slippage ในการตั้งราคาแก๊ส

💡 Why not Custom Gas Token?

OP Stack custom gas token ถูก deprecated/ถอดออก ดังนั้น ETH ยังเป็น gas token เสมอ — interop ระหว่างเชนจึงปลอดภัย ไม่แตกเป็นหลายหน่วยค่าแก๊ส Paymaster ทำ gas abstraction ที่ระดับแอป แทน จึงได้ UX จ่ายด้วย token โดยไม่แตะ gas token ของโปรโตคอล
